# ruby-cli-hello-world

## Description
This is a POC project to demonstrate
a classic ruby console app.

## Tech stack
- perl
- ruby 3.11

## Docker stack
- docker-cli
- alpine:latest
- centos:latest
- ubuntu:latest

## Requirements
Docker desktop must be installed and the application
being called must be linux compatiple.

## To run
```sudo ./install.sh -u```

## To stop (optional)
```sudo ./install.sh -d```
